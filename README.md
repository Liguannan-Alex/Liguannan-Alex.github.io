# 李冠南 · 个人主页（π·冠南）

生命科学极简风的单页个人网站：白底青绿基因视觉、双螺旋水印、A/T/C/G 履历时间轴，
首屏为粒子聚合头像入场（参考 sac-ai.com 的 particle-avatar 机制，纯原生 JS 实现，零依赖）。

**在线访问：https://liguannan-alex.github.io/**

## 文件

```
liguannan-portfolio/
├── index.html            # 全部内容 + 样式 + 动效，单文件
├── assets/
│   ├── avatar-cutout.png # 透明底卡通头像（展示图 + 粒子采样源）
│   ├── avatar.jpg        # 卡通头像原图（含天空背景，备用）
│   ├── headshot.jpg      # 真人照片（备用）
│   └── wechat-qr.png     # 公众号二维码 —— 需自行放入（见下）
└── README.md
```

## 本地预览

```bash
python3 -m http.server 8756
# 打开 http://127.0.0.1:8756
```

> 双击 `index.html` 用 `file://` 打开时，粒子入场动画会因浏览器安全策略自动降级为直接显示头像；其余功能不受影响。

## 放公众号二维码

1. 在微信后台 / 公众号页面导出「π·冠南」的二维码图片；
2. 命名为 `wechat-qr.png`，放进 `assets/` 目录，push 即可。

没放时会优雅降级成「π · 扫码 / 微信搜索 π·冠南」占位。

## 想改内容？

所有文字都在 `index.html` 的 `<body>` 里，按区块（hero / 关于 / 能力 / 履历 / 项目 / 代码 / 副业 / 联系）分好，直接改文字即可，样式（`<style>`）不用动。改完 push，Pages 一两分钟内自动更新。

## 响应式

≤560px 手机单列 / ≤900px 平板堆叠 / ≤1599px 常规桌面 / ≥1600px 大屏按视口比例整体缩放；
遵循 `prefers-reduced-motion`，无 JS 时内容完整可见。
